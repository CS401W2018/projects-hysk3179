Why do we need a viewport meta tag? : To mkae a web page is scaled and displayed on different screen sizes.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? : The last one in the code will be applied. It's like A= b; A= C; the A is C.

How do you change the CSS for a web page displayed on a phone? : By using media queries. @media we can use it.
