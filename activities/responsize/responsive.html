Responsive Pages

Why is the viewport meta tag important? - It controls how a web page is displayed on different devices, particularly mobile devices.

How can you adjust the CSS for printing? - print media query. for example - 'link rel="stylesheet" href="print.css" media="print"'

When would you want to adjust the CSS for printing? - Changing the layout, Optimizing for paper size
